python
======

A role for managing python install from package managers and virtualenvs.

Compatibility
-------------

  - CentOS 6
  - CentOS 7
  - Ubuntu 12.04
  - Ubuntu 14.04
  - Ubuntu 16.04
  - Debian 8
  - Debian 9
